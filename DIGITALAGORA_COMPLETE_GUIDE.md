# ЦИФРОВАЯ АГОРА – полное руководство по проекту (всё в одном файле)

## 1. О проекте

**DigitalAgora** – децентрализованная платформа для законотворческой деятельности акционеров (жителей).  
Позволяет:
- регистрироваться акционерам (фамилия, имя, отчество, дата рождения, место жительства, кошелёк, email, пароль);
- входить по email/паролю (JWT + bcrypt);
- создавать и обсуждать предложения по улучшению;
- голосовать токенами через смарт-контракт (Ethereum);
- анализировать идеи с помощью ИИ (OpenAI, заглушка);
- находить подрядчиков (демо-список);
- контролировать финансирование проектов.
