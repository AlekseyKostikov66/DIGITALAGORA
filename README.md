# DigitalAgora — Децентрализованная платформа управления ТСЖ

DigitalAgora позволяет жителям многоквартирных домов создавать предложения по улучшению, голосовать токенами, визуализировать проекты с помощью ИИ, автоматически находить исполнителей, контролировать выполнение работ и финансировать их поэтапно через смарт-контракты.

## Технологический стек

- **Блокчейн:** Hardhat, Solidity, OpenZeppelin (ERC20Votes, Governor, Timelock)
- **Хранение:** IPFS (через HTTP-клиент)
- **AI:** OpenAI GPT для анализа предложений, Replicate (Stable Diffusion) для визуализации
- **Бэкенд:** Node.js, Express
- **Фронтенд:** React, ethers.js, Axios

## Установка и запуск

# DigitalAgora

Децентрализованная платформа управления для товариществ собственников жилья (ТСЖ).

## Установка

1. Установить корневые зависимости: `npm install`
2. Скомпилировать контракты: `npm run compile`
3. Запустить локальный узел: `npm run node` (в отдельном терминале)
4. Развернуть контракты: `npm run deploy:local` (в отдельном терминале)
5. Запустить бэкенд: `npm run backend` (в отдельном терминале)
6. Запустить фронтенд: `npm run frontend` (в отдельном терминале)

## Окружение

- Бэкенд: `backend/.env`
- Фронтенд: `frontend/.env`

## Адреса

Смотрите `deployed.json` после развёртывания.
