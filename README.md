# ЦИФРОВАЯ АГОРА — платформа народного самодержавия

**ЦИФРОВАЯ АГОРА** является платформой для Законотворческой Деятельности каждого человека в режиме реального времени 24/7, что является краеугольным камнем **НАРОДНОГО САМОДЕРЖАВИЯ**.

> **ГАО "ИИСУС ХРИСТОС" — Компромисс Добра со злом! И волки сыты, и овечки целы!**

DigitalAgora позволяет жителям многоквартирных домов создавать предложения по улучшению, голосовать токенами, визуализировать проекты с помощью ИИ, автоматически находить исполнителей, контролировать выполнение работ и финансировать их поэтапно через смарт-контракты.

## Технологический стек

- **Блокчейн:** Hardhat, Solidity, OpenZeppelin (ERC20Votes, Governor, Timelock)
- **Хранение:** IPFS (через HTTP-клиент)
- **AI:** OpenAI GPT для анализа предложений, Replicate (Stable Diffusion) для визуализации
- **Бэкенд:** Node.js, Express, MongoDB, bcrypt, JWT
- **Фронтенд:** React, ethers.js, Axios
