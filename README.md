# ЦИФРОВАЯ АГОРА — децентрализованная платформа народного самодержавия

**ЦИФРОВАЯ АГОРА** является платформой для Законотворческой Деятельности каждого человека в режиме реального времени 24/7, что является краеугольным камнем **НАРОДНОГО САМОДЕРЖАВИЯ**.

> **ГАО "ИИСУС ХРИСТОС" — Компромисс Добра со злом! И волки сыты, и овечки целы!**

DigitalAgora позволяет жителям многоквартирных домов создавать предложения по улучшению, голосовать токенами, визуализировать проекты с помощью ИИ, автоматически находить исполнителей, контролировать выполнение работ и финансировать их поэтапно через смарт-контракты.

---

## 📌 Текущий статус

- ✅ **Бэкенд** (Node.js, Express, JWT, bcrypt) – регистрация, вход, хранение пользователей в JSON-файле.
- ✅ **Фронтенд** (HTML/CSS/JS, без React) – дашборд с разделами: Предложения, ИИ-анализ, Подрядчики, Профиль.
- ✅ **Блокчейн** (Hardhat, смарт-контракты ERC20Votes + Governor) – локальная сеть, создание предложений и голосование через MetaMask.
- ⚠️ **В разработке**: чтение предложений из блокчейна, делегирование голосов, интеграция реального ИИ.

## 🧱 Архитектура
[Браузер] → http://localhost:3000 (статический фронтенд)
│
├─ REST API → http://localhost:5000 (бэкенд)
│ ├─ /auth/register, /auth/login
│ └─ сохранение профилей в data/users.json
│
└─ Блокчейн → http://localhost:8545 (нода Hardhat)
├─ AgoraToken (ERC20Votes)
└─ AgoraGovernor (голосование)



**Технологии**: Node.js, Express, bcrypt, JWT, ethers.js, Hardhat, Solidity (OpenZeppelin), HTML/CSS/JS.

## 🚀 Быстрый старт (для разработчиков)

### 1. Установка зависимостей


# Корень (Hardhat)
npm install

# Бэкенд
cd backend
npm install express jsonwebtoken bcrypt dotenv cors
cd ..

2. Настройка окружения
Создайте backend/.env:

ini
PORT=5000
JWT_SECRET=суперсекретный_ключ_12345
3. Запуск компонентов (4 терминала)
Терминал 1 – блокчейн-нода


npx hardhat node
Терминал 2 – деплой контрактов


npx hardhat run scripts/deploy.js --network localhost
Скопируйте адреса контрактов из deployed.json.

Терминал 3 – бэкенд

cd backend && node server.js
Терминал 4 – фронтенд (статический сервер)

cd frontend/public && npx http-server -p 3000

4. Открыть в браузере
Перейдите на http://localhost:3000

Зарегистрируйтесь (все поля обязательны, кроме отчества).

Войдите.

Подключите MetaMask к сети http://localhost:8545 (Chain ID 31337). Импортируйте тестовый аккаунт (ключи видны при запуске npx hardhat node).

Создавайте предложения и голосуйте.
